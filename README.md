# ThirdPwesonShooter 🎮
Unreal Engine으로 제작한 3인칭 슈팅 게임 프로젝트입니다.

## 📺 시연 영상
[![Youtube Video](https://img.youtube.com/vi/영상ID/0.jpg)](https://www.youtube.com/watch?v=영상ID)  
→ 이미지를 클릭하면 유튜브 영상으로 이동합니다.

## 📌 프로젝트 소개
- 제작 기간: 2025.03 ~ 2025.06
- 사용 엔진: Unreal Engine 5.3
- 개발 언어: C++

## 📝 주요 기능
- 3인칭 슈팅 시스템 구현
- 기본 전투 및 이동 시스템
- 네트워크 멀티플레이 지원
